# DBSCAN Clustering Analysis

## Project Overview
This project demonstrates the implementation of the DBSCAN (Density-Based Spatial Clustering of Applications with Noise) algorithm.
DBSCAN is an unsupervised learning technique used to identify clusters of varying shapes and detect noise/outliers in data.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Algorithm Used
- DBSCAN (Density-Based Spatial Clustering)
